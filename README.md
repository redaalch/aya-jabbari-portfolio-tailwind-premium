# Aya Jabbari Portfolio

A production-ready Vite, React, and Tailwind CSS portfolio for Aya Jabbari, focused on MERN stack projects, internship experience, and AI/document automation work.

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS v4 for all styling
- Motion for subtle section animation
- Lucide React for icons

No shadcn/ui or Radix UI component library is used. The layout and components are custom React components styled directly with Tailwind utilities.

## Getting started

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Project structure

```txt
src/
  assets/       Resume and local profile image
  components/   Reusable page sections
  data/         Portfolio content in one place
  styles/       Tailwind and global styles
```
